¿Qué es Flex Box?

El módulo de caja flexible, comúnmente conocido como flexbox, se desarrolló como un modelo de diseño unidimensional y es una técnica que puede ayudar a organizar el espacio entre los elementos de una interfaz y mejorar la alineación. Este artículo cubre las funciones básicas de Flexbox, que veremos con más detalle en el resto del tutorial. Cuando describimos a flexbox como unidireccional, enfatizamos el hecho de que flexbox solo maneja el diseño en una dimensión a la vez: fila o columna. Esto contrasta con el modelo 2D de diseño de cuadrícula CSS, que controla columnas y filas al mismo tiempo.
